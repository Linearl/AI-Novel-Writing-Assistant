category: 修辞
---

## 核心原理

人脑对重复出现的信息会自动启动"省电模式"——熟悉的画面、词语和叙事会被快速略过而不被真正感知。陌生化正是对抗这种认知惰性的武器：它把习以为常的人、事、景物、情绪拆解开来，用全新的视角和逻辑重新组装，让熟悉的事物重新变得新鲜、刺眼、不可忽视。陌生化不是堆砌华丽辞藻，而是改变观察事物的角度——当"日出"被写成"黑夜慢慢卸下沉重的衣裳"，读者被迫停下来，用全新的眼光重新审视这个每天都在发生的事件，由此被唤醒的感知力远超任何新奇的内容本身。

## 使用方法

1. 颠倒常规视角：跳出惯性认知，从被忽略的角度切入（从景物看人、从结果看起因）
2. 反常搭配用词：拒绝最直接的形容词，选择错位的动词和名词搭配（"夜晚浸着凉意"而非"夜晚很凉"）
3. 拉长细节感知：放慢叙事节奏，将一个瞬间拆解成多个微小的变化过程
4. 陌生化的尺度要适中——过于陌生变成晦涩，不够陌生则失去效果

## 示例

### 示例一：写春天草木萌发

- 原句：春天来了，小草从土里长了出来
- 改写：小草偷偷地从土里钻出来，嫩嫩的，绿绿的，像憋了一整个冬天的心事终于憋不住了

### 示例二：写深夜失眠

- 原句：他躺在床上睡不着，翻来覆去到了天亮
- 改写：床单慢慢记住了他的形状，时钟的秒针走得像在拖拽什么重物，天花板在黑暗中一寸一寸地后退

### 示例三：写重逢时的物是人非

- 原句：多年后回到老家，发现一切都变了，很感慨
- 改写：门前的路认出了他的脚步，但门牌已经换了姓氏——他在自家门口成了一个需要被介绍的陌生人

## 关键提示

陌生化不等于新奇比喻——普通比喻是常规联想的延伸，陌生化是跳出固有认知框架，让读者重新"看见"本以为早已熟悉的一切。
