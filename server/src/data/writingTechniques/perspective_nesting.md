## 核心原理

视角嵌套法就是在同一个场景中设置两个主体，通过"你映衬我，我成就你"的视角转换，让双方互为风景、互为装饰。如同运用不同的运镜方式切换视角——从近镜头切换到远镜头，或在同一个视角高度切换主体。

它打破了单向叙事的单薄感：不是"我在看风景"，而是"我在看风景，看风景的人也在看我"。读者从不同视角看到同一件事的全貌，情绪不是被告知的，而是在双向映照的细节中自己品出来的。

## 使用方法

1. **建立双向视角的思维习惯**：写下A和B的场景时，停下来问自己——这个时候B在干什么？对A是什么感觉？多加这一问，嵌套的雏形就出来了
2. **用拉视角制造惊喜**：先写近景（A看B），再拉远写远景（B看A），或从写实镜头切到写意升华
3. **保持双向性是灵魂**：必须有两个及以上主体，且彼此互为观察者、互为素材
4. **结构上形成闭环**：让内容形成一个完整的循环，读者读完后有穿透的惊喜感

## 示例

### 示例一：卞之琳《断章》

- 原句：我在桥上看风景，风景很美
- 改写：你站在桥上看风景，看风景的人在楼上看你。明月装饰了你的窗子，你装饰了别人的梦。

### 示例二：迟子建《带笤帚的小鸟》

- 原句：大雪天，母亲在窗前看鸟，觉得很温馨
- 改写：我想小鸟儿在户外望着那些花时，一定很疑惑：这家人，大雪天的，怎么过着春天的日子呢？鸟儿赏花的时候，母亲也在窗前悄悄赏它们。它们在不经意间，也成了她眼里的春色了！

### 示例三：写图书馆里的两个人

- 原句：我在图书馆看书，对面有个女生也在看书
- 改写：她翻书的时候很轻，像怕把某个沉睡的字吵醒。我不知道她读到哪里了，只看见她的眉头偶尔皱一下，嘴角偶尔翘一下。我猜她在看一本结局不太好的小说。后来我低头继续读自己的书，余光里她的影子晃了晃——大概是伸了个懒腰。再抬头时，她正看着我。我愣了一下，她也愣了一下。然后我们都笑了。原来她一直在看我读她正在读的那本书，而我一直在偷看她翻书的样子。窗外的阳光移了移位置，把我们两个人的影子投在同一张桌上，像两页被风吹到一起的书。

## 关键提示

最忌硬凑主体——两个主体毫无关联时强行绑定会特别生硬，读起来像毫无逻辑的连线题。呼应的最高境界是"藏"——写完句子就好，千万别在后面加一句"这就是双向呼应"，说破了意境就散了。一篇文章用一两次作为点睛之笔就够了。
