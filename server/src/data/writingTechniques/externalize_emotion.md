## 核心原理

最私人的情绪往往需要通过最公共的感官来传递。当你说"心痛"读者知道但不一定感受得到，当你说"心像被一只湿冷的手攥住了"，每个曾在夜里无助过的人都会瞬间懂得。大脑在进化中总是先相信感官，然后才理解情绪。

## 使用方法

1. 触觉嫁接法：捕捉情绪在身体上的反应，寻找类似的触感来外化
2. 色彩转码法：给情绪穿上颜色的衣服，用颜色的质变暗示情绪的变化
3. 通感编织法：让情绪在感官之间流动，用一种感官描述另一种感官的体验
4. 解剖句法：不写笼统的"我"，只写失控的身体——脑子、嘴、手、眼睛、心脏各干各的，情绪张力直接拉满。公式：嘴上说XX + 身体在XX + 动作失控。少用"我"，多用手、眼、喉咙、心脏、脊背；造反差（嘴上逞强，身体诚实）；加触感（凉、僵、疼、抖、沉比情绪词更戳人）

## 示例

### 示例一：被抛弃的空虚

- 原句：他走后她觉得非常空虚，好像被世界抛弃了
- 改写：他走后第三天，沙发上的抱枕还保持着他靠过的形状，我伸手去摸，绒布已经凉透了

### 示例二：青春消逝

- 原句：回忆起青春岁月，她觉得一切都变了，青春一去不复返
- 改写：记忆里的夏天原本是明黄色的，如今却像旧报纸，边角泛黄，字迹模糊

### 示例三：复杂的心情

- 原句：听到那个消息她又惊又喜，心情很复杂，说不清是什么滋味
- 改写：那个消息像不小心嚼碎的花椒，麻味过后，才是挥之不去的涩

### 示例四：纠结的撕裂

- 原句：我很纠结，舍不得走
- 改写：理智催我转身，腿却像钉在原地，手在发抖，眼睛还拼命想多看一眼

### 示例五：隐忍的委屈

- 原句：她心里很委屈，但强忍着没哭
- 改写：笑着说"我没事"，鼻子一酸，眼眶发烫，转身才敢红了眼

## 关键提示

下次写情绪时问自己：这种情绪摸起来是什么手感？尝起来是什么味道？看起来是什么颜色？
