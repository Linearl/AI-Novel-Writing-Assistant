## 核心原理

读者记住的从来不是"完美角色"，而是"矛盾角色"。人物反差的本质是让角色在不同情境下展现出截然不同的一面，制造认知落差。这种落差激活读者的好奇心——"他怎么会这样？"——从而产生深入阅读的动力。反差越大，角色越立体，读者越沉迷。

## 使用方法

### 维度一：外表X内在反差

最经典的反差类型。外在形象给读者一个预期，内在性格给出完全相反的答案。

- **外冷内热**：冷硬外表下藏着柔软内心
- **外热内冷**：温和笑脸背后没有温度

### 维度二：人前X人后反差

同一角色在公共场合和独处时展现两副面孔，揭示社会面具下的真实自我。

- **人前完美 X 人后脆弱**：职场精英独处时崩溃
- **人前社恐 X 人后疯批**：现实中唯唯诺诺，虚拟世界杀伐果断

### 维度三：言行X举止反差

语言表达和行为模式之间的矛盾，暗示角色的复杂性格。

- **毒舌X护短**：嘴上不饶人，行动上第一个挡在前面
- **寡言少语X细节控**：话不多说，却把你说过的每句话都记在心里

### 维度四：身份X行为反差

社会地位、职业身份与私下行为之间的落差，制造意外感和亲和力。

- **身份尊贵X接地气**：高高在上的人物有接地气的小癖好
- **清冷学霸X反差萌点**：完美人设背后藏着可爱的小秘密

### 维度五：过去X现在反差

同一角色在不同时期的巨大转变，暗示成长弧线和命运走向。

- **曾经桀骜X如今温顺**：浪子回头的治愈弧线
- **曾经怯懦X如今强势**：弱者蜕变的爽感弧线

## 示例

### 示例一：外冷内热

- 原句：她虽然看起来很冷，但其实内心很善良。
- 改写：她穿着一身冷硬的黑色皮衣，眉眼锋利如刀，说话也总是带着刺，可每次巷口的流浪猫出现，她都会从包里掏出温热的罐头，蹲在地上轻声哄着，指尖动作温柔得不像她。

### 示例二：人前完美 X 人后脆弱

- 原句：他在公司很坚强，回家却很脆弱。
- 改写：在职场上，他永远是行事果决的部门主管，应对客户从容不迫，共事许久的同事，也从未见过他失态半分。可回到空荡冷清的公寓，他随手将西装外套甩在沙发上，顺着墙面滑坐在地，指尖攥着早已过期的药盒，眼泪无声地落了下来。

### 示例三：毒舌X护短

- 原句：他平时爱欺负她，但不允许别人欺负她。
- 改写：他平日里总爱怼她，笑她路痴、嫌她马虎，连她做的饭菜都能挑出百般不是。可一旦旁人说她半句不好，他第一个沉脸发作，将她护在身后，冷声道："我怎么说她都行，还轮不到你来指手画脚。"

### 示例四：身份尊贵X接地气

- 原句：虽然是总裁，但私下很随和。
- 改写：他是人人敬畏的集团总裁，出入都是保镖随行，可私下里却最爱蹲在路边的小摊吃烤串，连老板都认识他，每次都笑着给他多撒点辣椒，他也不摆架子，接过串就咬，吃得一脸满足。

### 示例五：曾经桀骜X如今温顺

- 原句：他以前很叛逆，现在变得很温和。
- 改写：他以前是学校里的问题少年，打架逃课样样精通，谁都管不住他。可现在他穿着白大褂，戴着金丝眼镜，说话轻声细语，对着病人耐心解释病情，连眼神里都带着温和。

## 关键提示

1. **反差要通过具体行动展现**，不要直接说"他其实很善良"——让读者自己从行为中读出反差。
2. **反差点要服务于角色弧线**，不是为了反差而反差，每一次反转都应该推动读者对角色的理解。
3. **同一角色可叠加多个维度的反差**，但要确保核心人格统一，反差是"同一灵魂的不同切面"。
4. **用细节锚定反差**：一罐温热的猫粮、一个粉色手机壳、一盒剥好的栗仁——这些具体细节比任何形容词都有力。
