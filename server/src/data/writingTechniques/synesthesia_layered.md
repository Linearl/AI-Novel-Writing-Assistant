## 核心原理

单纯使用五感法写出的文字有画面感，但往往质感不足。读者会觉得"确实有画面，但我自己也能写"。兼格是指在同一语言表达中，两种或多种修辞格自然融合、同时发挥作用。它通过巧妙组合不同修辞的优势，让语言在形象、情感、逻辑上形成多重张力，实现"一箭多雕"的文学效果。

实践中最出彩的公式是：**通感为主格 + 比喻/排比为副格**。通感打通感官壁垒，比喻提供形象支点，排比制造节奏势能，三者叠加时文字的密度和张力远超单一修辞。

## 使用方法

1. **先完成五感描写**：把场景中的视觉、听觉、嗅觉、味觉、触觉写出来
2. **叠加通感**：选择其中一种感官，用另一种感官来描述它（视觉→触觉、听觉→味觉等）
3. **叠加比喻**：为通感中的关键意象加上比喻，从静态到动态层层递进
4. **叠加排比**：用排比结构制造节奏感，引导读者感官一步步沉浸
5. **自检三问**：这个五感可以"像"什么吗？可以"发生"什么吗？出声读一遍能感受到画面质感吗？

## 示例

### 示例一：朱自清《绿》——描写梅雨潭的绿

- 原句：梅雨潭的绿很醉人，很美
- 改写：那醉人的绿呀！她松松地皱缬着，像少妇拖着的裙幅；她轻轻地摆弄着，像跳动的初恋的少女的心；她滑滑地明亮着，像涂了明油一般，有鸡蛋清那样软，那样嫩，令人想着所曾触过的最嫩的皮肤。

### 示例二：朱自清《荷塘月色》——描写荷香

- 原句：微风送来荷花的清香，很好闻
- 改写：微风过处，送来缕缕清香，仿佛远处高楼上渺茫的歌声似的。

### 示例三：余华《活着》——描写月光与思念

- 原句：月光照在路上，我很想念儿子
- 改写：我看着那条弯曲着通向城里的小路，听不到我儿子赤脚跑来的声音，月光照在路上，像是撒满了盐。

### 示例四：描写歌声（原创练习）

- 原句：她的歌声悠悠入耳，很治愈
- 改写：她的歌声甜得像浸了蜜的月光，顺着耳廓滑进他灰暗的眼眸，在瞳孔里凝成了晶莹的光。

## 关键提示

兼格不是堆砌——每增加一种修辞都必须服务于核心意象，不能为了炫技而叠加。通感是主格，负责打通感官壁垒；比喻和排比是副格，负责加厚质感和节奏。如果五感本身不够扎实，叠加再多修辞也只是空中楼阁。
