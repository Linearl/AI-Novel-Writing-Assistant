## 核心原理

不喊不叫不哭不闹，平静地写一个人最平常的生活细节，但读完心里堵得慌。这种委屈不是爆发式的，是慢慢渗透进生活缝隙里的——像南方的梅雨天，空气一直是潮的。把哭声按回喉咙里，转化成一个气味、一个动作、一朵清淡的花。

## 使用方法

1. 用气味承载屈辱：不说"我很委屈"，只写从日常中散发出来的某种挥之不去的味道
2. 用动作隐藏心碎：不说"他很自卑"，写他在日常中那些下意识的退缩和偷望
3. 用物件见证沦陷：找一个日常物件，让它的渐变对应人物命运的沉浮
4. 用日常习惯写负罪感：不说"他很愧疚"，写他在日常动作中流露出的那份小心翼翼

## 示例

### 示例一：自卑的凝视

- 原句：他觉得自己配不上她，总是远远地看着她
- 改写：我躲在船舱里偷偷地看她，我觉得她就是那个向日葵，永远向着太阳，那么明亮。而我呢，我就是水里的一根烂水草

### 示例二：被遗忘的老人

- 原句：老人被子女遗弃了，每天一个人吃饭，非常孤独
- 改写：每天傍晚她都把两双筷子摆在桌上，摆好了愣一下，又收回去一双。然后低头吃饭，一口一口，嚼得很慢

### 示例三：贫寒中的体面

- 原句：家里虽然很穷，但她依然努力维持体面，把最好的留给孩子
- 改写：她用最后一点面粉蒸了馒头，把最白的那个放在孩子碗里，自己拿起那个蒸裂了的，对着灯光看了看，掰开吃了

## 关键提示

文字不是用来喊的，是用来渗的——把"很委屈"三个字删掉，换成一个人在委屈时会做的事。
