## 核心原理

落点句的核心机制是"蓄力—释放"。前半段故意写得松散、平淡，甚至像闲聊和流水账，让读者的注意力和戒备心都慢慢松懈下来。当最后一句突然以一个精准的物件或画面收网时，前面所有看似无关的细节瞬间被重新激活，每一句平淡的话都变成了为这一击蓄积的力量。落点句不同于伏笔：伏笔是提前埋下线索等读者日后发现，落点句是当场引爆——前面散得越彻底，后面收得越有力。它的底层逻辑是利用"预期落差"：读者以为你在闲聊，你却在蓄力。

## 使用方法

1. 前文故意写得松散、缓慢，甚至带一点"废话感"，让读者以为你在铺垫日常
2. 中间可以加入看似无关紧要的物件或细节，它们是蓄力的引线
3. 最后一句突然收网：用一个具体的物件（越小越好）、一个"还在"的意象、或一个极轻的动作代替大段抒情
4. 收网后不要再补充任何解释，让读者自己在沉默中体会

## 示例

### 示例一：空房子

- 原句：他回到老家，发现房子已经空了，心里很难过
- 改写：墙皮掉了一块，灶台塌了半边，院子里长了半人高的草。他蹲下来摸了摸门槛上的裂纹，抬头看见房梁上，去年燕子搭的窝还在。

### 示例二：散伙饭

- 原句：毕业散伙饭那天，大家都很开心，但分别的时候还是很难过
- 改写：菜上齐了，酒倒满了，谁也没先提散的事。张伟讲了个冷笑话，全桌笑了，他笑得最大声。最后结账的时候，老板说八百六，他掏出手机付了，然后问了一句：以后还来吗？

### 示例三：母亲的手

- 原句：妈妈老了，手上都是皱纹，看着很心疼
- 改写：她剥了一整盘蒜，蒜皮堆在桌角像一座小山。我注意到她的手背上有好几道裂口，但她一直没停下来。蒜剥完了，她把盘子推到我面前说：够了吗？

## 关键提示

别怕散。前面所有的"轻"，都会在最后一秒变成"重"。
