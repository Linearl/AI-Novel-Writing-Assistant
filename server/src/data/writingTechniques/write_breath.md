## 核心原理

好的文字会呼吸。节奏、停顿、轻重缓急形成看不见的"气息"，牵着读者心跳走完全篇。气息是先于语言的生理反应——紧张节奏让你不自觉屏息，舒缓语流让你心神放松。控制气息就是直接控制读者的阅读体验。

## 使用方法

1. 用标点控制呼吸点：句号是停顿，逗号是换气，顿号是急促连续，省略号是悠长余韵
2. 用句长调节呼吸节奏：重要需要强调的事用短句如鼓点，描写氛围绵长感受用长句如长笛
3. 用词语模仿呼吸状态：写慌乱用碎词短句，写平静用叠词长句

## 示例

### 示例一：恐惧场景

- 原句：他非常害怕，在黑暗中听到脚步声越来越近
- 改写：心，咚咚咚。撞着胸口。黑暗中，只有这个声音，和门外越来越近的……脚步声

### 示例二：悠闲午后

- 原句：午后的阳光照进来，房间很温暖很舒服
- 改写：阳光斜斜地挪过窗台，茶杯里的热气懒懒地打着旋儿，一本翻开的书，在膝上睡着了

### 示例三：争吵升级

- 原句：两个人越吵越凶，最后完全失控了
- 改写：她说了第一句。他回了第二句。第三句还没出口，杯子砸在地上——碎片溅开的声音，把后面所有的话都盖住了

## 关键提示

气息是所有技巧的底层逻辑，真正厉害的作家未必是词汇的富翁，但一定是掌控读者呼吸的节奏大师。
