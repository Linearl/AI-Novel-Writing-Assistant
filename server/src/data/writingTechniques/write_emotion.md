## 核心原理

"开心、难过、紧张"是情绪的冰冷标签，是结论不是过程。人的情绪发生在一分一秒里的身体反应和心理活动中——把它慢镜头回放，拆解成可感知的生理与行为颗粒，读者就能进入你的心跳、你的胃、你一瞬间的恍惚，去亲历而非听说。

## 使用方法

1. 定位情绪的"生理坐标"：问自己这种情绪在身体哪里、什么感觉
2. 捕捉"瞬间行为"：情绪泄露在不经意的小动作里，抓住那个犹豫或失控的瞬间
3. 借用环境"感官投射"：把情绪投射到环境感知中，让外部世界染上内心的颜色

## 示例

### 示例一：心动瞬间

- 原句：她心里小鹿乱撞，觉得很甜蜜
- 改写：接过那杯奶茶时，她的指尖不小心碰到了他的。接下来整整十分钟，那块皮肤都像被阳光灼过一样，残留着一圈清晰的、发烫的触感

### 示例二：极度恐惧

- 原句：他吓得浑身发抖，害怕极了
- 改写：他的膝盖撞在了一起，牙齿在打颤，舌头尝到了铁锈味——他咬破了口腔内壁，自己却毫无察觉

### 示例三：强撑之后

- 原句：她一直忍着不让眼泪掉下来，终于忍不住了
- 改写：她的喉结上下滚动了两次，鼻腔里突然涌上一股酸涩，眼眶发烫，她猛地抬头看向天花板，拼命把那股热意逼回去

## 关键提示

三个心法：写反常不写寻常，写过程不写结果，写私有不写公有——越私有的细节反而越能引发普遍共鸣。
