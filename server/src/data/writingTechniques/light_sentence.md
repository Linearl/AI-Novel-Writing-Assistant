## 核心原理

光线句不写物体长什么样，而是写光照在它上面的样子。光是视觉叙事中最有力的工具之一：光打在哪里，读者的注意力就跟到哪里；光的冷暖暗示着人物的心理温度；光的明暗分割出可见与不可见的边界。同样的房间、同一张脸，侧光和逆光传达的情绪截然不同。光线句的底层逻辑是把光当作情绪的代理语言——不需要说"这个人很孤独"，只需要写"光只照到他一半的脸，另一半陷在黑暗里"，读者就能自己感受到那种被世界遗弃的落寞。

## 使用方法

1. 用光切割空间：制造亮和暗的交界线，让光照亮一部分、遮蔽一部分，形成情绪的张力
2. 用光打在物体表面创造质感：光打在不同材质上效果不同，选择能暗示情绪的那一种
3. 用光的温度代替心理描写：冷光暗示冷漠和疏离，强光暗示崩溃和无法躲藏
4. 用光影的移动暗示时间流逝：不需要写"过了很久"，写光从桌面移到地上就够了

## 示例

### 示例一：失眠的夜晚

- 原句：她失眠了，在黑暗中想了很多事情，感到很孤独
- 改写：她没有开灯，窗帘缝里漏进来一道月光，刚好落在床单上他那边的位置。她盯着那道光看了很久，直到它慢慢移到了地上。

### 示例二：面试失败后

- 原句：他面试失败了，从写字楼里出来，心情很差
- 改写：写字楼的玻璃门把他的影子拉得很长。他走进地铁站的时候，日光灯把他的影子突然切短了。他低头看了一眼脚边那个缩小的自己，加快了脚步。

### 示例三：深夜医院走廊

- 原句：她在医院守了一整夜，非常疲惫和担忧
- 改写：走廊的灯管嗡嗡响着，惨白的光照在她的手背上，青色的血管一根一根看得清楚。她把袖子拉下来盖住手，但灯光穿过了布料。手术室的门"咔"地响了一声，灯还是白的。

## 关键提示

你不只是编剧，也是灯光师——问自己：这一幕，光打在哪里最扎心？
