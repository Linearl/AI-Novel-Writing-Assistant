## 核心原理

海明威冰山理论在对话中的应用：人物说出口的只有八分之一，剩下的八分之七藏在动作、沉默和答非所问里。读者自行补全的部分，比直接说出来更有力量。因为人的大脑天生厌恶空白，一旦对话留出缺口，读者会本能地用自己的经验去填充，而每个人填进去的东西都带着自己的情感投射——于是同一段话，一千个读者读出一千种痛。克制是冰山对话法的核心美德：你说得越少，读者感受到的越多。

## 使用方法

1. 让人物答非所问——不直接回答问题，用别的话回避，回避本身就是最诚实的回答
2. 用动作打断对话——话说到一半，让人物做一个动作代替说下去，让身体接管语言
3. 关键那句话让沉默来说——不让人物自己说出最重要的话，用停顿或环境音代替
4. 让重复暴露真相——同一句话在不同语境下出现两次，含义天翻地覆

## 示例

### 示例一：争吵后的沉默

- 原句："我还在生你的气，因为你昨天当着所有人面让我难堪了。"
- 改写："还在生气？"她没说话，把他的拖鞋从门口收回鞋柜，放在自己那双旁边。放完了，又拿出来，放到了门边。（一个放鞋的动作，收回又拿出，比任何解释都更精确地画出了她内心的拉锯。）

### 示例二：离别的克制

- 原句："我很舍不得你走，但我知道这是对的，希望你一路顺风。"
- 改写："到了给我发消息。""知道了。""别老熬夜。""嗯。""……锅里还有汤，你喝完再走。"（最后那句汤，才是她真正想说的话。前面所有叮嘱都是在拖延开口说"走"的时刻。）

### 示例三：谎言的裂缝

- 原句："他说他没有做过那件事，但其实他心虚极了。"
- 改写："钱呢？"他看了她一眼，又看了窗外一眼。"花了。""花在哪了？"他开始整理桌上散落的纸牌，把它们一张一张码齐。"不记得了。"（他不看她，他看窗户，他码纸牌——身体在疯狂地逃跑，嘴上只说了三个字。）

## 关键提示

冰山对话法是做减法：删掉解释，藏起情绪，把形容词交给动作和沉默。越克制，读者感受到的情感越重。
