## 核心原理

全篇构建严丝合缝的闭环——开篇的画面、动作、意象在结尾完整复刻折返。起始即是落幕，落幕又重启起始，如同蛇首尾相衔死死咬住。故事读到末尾，线索重新拽回开篇，没有真正的终点，只剩无限重复的宿命轮回。用文字画一道闭环枷锁，把人物与情绪锁在往复的圈子里无处脱身。

## 使用方法

1. 首尾意象精准扣合：开篇核心景物、动作、台词，结尾原模原样对接，一字一景严丝合缝
2. 催生重读冲动：读者读完下意识回头翻看开头确认轮回闭环——这份求证心思正是手法生效的证明
3. 循环承载内核情绪：不为炫技刻意绕圈，闭环恰好匹配主题——执念、煎熬、无解的遗憾、逃不开的重复命运

## 示例

### 示例一：写无望等待

- 开头：他又坐在靠窗老位置，面前咖啡早已放凉，一口未动
- 结尾：指尖碰了碰冰凉的杯壁，终究没有喝下，轻轻推回桌面。明日这个时辰，他依旧会坐在这里

### 示例二：写困顿日常

- 开头：六点十五分，闹钟准时响起，他抬手按灭，缓缓坐起身
- 结尾：双眼合上入睡，心知天亮六点十五分的闹钟，依旧会如期响起

### 示例三：写失眠

- 开头：天花板三道细纹，他翻来覆去数了无数遍
- 结尾：强行闭眼片刻再睁眼，天花板依旧是那三道细纹，长夜没有尽头

## 关键提示

专门适配写执念、旧伤、难解的心结、反复犯错、一成不变的困顿人生。不必直白堆砌"永远"二字——让闭环自行轮转，一圈便是漫长煎熬，圈圈叠加便是无解的轮回。
