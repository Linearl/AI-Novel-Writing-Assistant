## 核心原理

汉语的词类本身就具有弹性，一个词在不同语法位置上可以自由切换身份，这种灵活性为转品提供了天然土壤。当一个词脱离它最常见的词性位置，被放置到陌生的句法环境中时，读者的自动阅读模式会被打断——原本一扫而过的信息突然需要停下来细品。这个短暂的"愣"就是文学性的起点：名词化为动作时获得了力量，形容词化为动作时获得了画面，意象直接嫁接质感时获得了通感般的冲击。转品用最小的改动撬动最大的表达升级。

## 使用方法

1. 找到一个有物理触感或画面感的名词，把它拽进动词的位置（如"愧疚烙了他一辈子"）
2. 选取一个形容词，让它承担施加力量的动词功能（如"这场寒流冰住了整条河"）
3. 用名词直接替代形容词的位置，让意象本身携带修饰功能（如"这个小镇的味道很旧时光"）
4. 控制转品的频率——一句一转品即可，密度太高会丧失冲击力

## 示例

### 示例一：写长期的悔恨

- 原句：他对那件事非常后悔，这份后悔伴随了他很多年
- 改写：愧疚烙了他一辈子，走到哪儿都带着那道看不见的伤疤

### 示例二：写冬夜的寒意

- 原句：冬天的寒风刮了一整夜，河面都结了冰
- 改写：这场寒流冰住了整条河，连船的声音都被冻在了水面下

### 示例三：写故乡老街的味道

- 原句：那条老街让人想起过去的日子，有一种怀旧的感觉
- 改写：巷口那家铺子的气味很旧时光，闻一下就把人拽回了童年

## 关键提示

好的转品会改变一个词在读者心中的永久位置，之后再看到它的常规用法，总觉得少了什么。
