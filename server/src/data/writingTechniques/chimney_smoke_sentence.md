---
name: 炊烟句
description: 用日常不变的小细节写时间流逝和人事变迁，以"还在"或"不在"制造反差
category: 句法
---

## 核心原理

炊烟句用"不变"来写"万变"。世界每天都在变，但总有一些微不足道的小细节被时间遗忘——灶台上积的灰、门口歪了的信箱、墙角那棵没人浇水却还在长的野草。当"东西还在"而"人已不在"时，这种反差不需要任何抒情就已经足够伤人。炊烟句的底层逻辑是利用人类对"日常感"的依恋：读者不需要理解抽象的"失去"是什么意思，只要看到那个具体的东西"还在"，就会本能地感受到背后那个"不在"的人留下的空洞。

## 使用方法

1. 选择一个具体而微小的日常物件，越不起眼越好（灶台、门锁、阳台灯、旧书签）
2. 用一种写法呈现：它还和从前一样（暗示人已不在）；它不一样了（呈现变化）；它还在，但你看不见了（写离开的人）
3. 不解释情绪，只提供一个客观证据：物件的状态 + 人的缺席
4. 不要写"我很想念"之类的话，让"那个东西还在"这句话自己替你说完

## 示例

### 示例一：外婆的灶台

- 原句：外婆去世后，我非常想念她，常常想起她在厨房做饭的样子
- 改写：外婆走后，灶台再也没有热过。锅盖上落了一层灰，灰上面有一个指印，是我上次来时碰的。

### 示例二：异地恋的结束

- 原句：分手以后，我看到我们以前一起去过的地方就很难过
- 改写：那家面馆还在，门口多了一块"旺铺招租"的牌子。老板娘看见我，还是问了一句：老样子？我说，嗯。面端上来的时候我才发现，我已经不吃香菜了。

### 示例三：老屋拆迁

- 原句：老家拆迁了，我很舍不得那些童年回忆
- 改写：推土机停在院子外面，铁门上的对联还是我小学三年级写的，墨迹淡得快看不见了。"年年有余"的"余"字歪在一边，但没有掉下来。

## 关键提示

每一个"再也不会"的背后，都站着一个曾经"一直会"。
