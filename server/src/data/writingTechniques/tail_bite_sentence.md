## 核心原理

咬尾句让每个句子的结尾成为下一句的开头，像多米诺骨牌一样一环扣一环地推着读者往前走。同一个词在连续的句子中反复出现，但身份不断变化——第一次是宾语，第二次变成主语；第一次是静态的物，第二次变成了动态的参与者。这种结构制造出一种宿命般的推进感，读者被同一个词钩住，不由分说地被拽向下一个句子、再下一个句子。咬尾句的底层逻辑是利用"重复即强化"：同一个词每出现一次，它的意义都叠加了一层新的重量，直到最后那个闭环的句号落下时，整个链条的情绪已经积累到了顶点。

## 使用方法

1. 选择一个名词作为钩子——名词比动词稳定，比形容词具象，更容易承载身份转换
2. 让钩子每次出现时换一个身份：从被描述的对象变成动作的发出者，从背景变成主角
3. 控制链条长度，三到六句最佳，太短没有推进感，太长会失去张力
4. 最后一句让钩子闭合——要么回到起点，要么走向虚无，形成一个完整的情绪环

## 示例

### 示例一：雨夜独行

- 原句：下雨了，她一个人走在空荡荡的街上，心里很乱
- 改写：雨落在窗台上。台上积了一小洼水。水里映着对面楼的光。光里站着一个模糊的人影。人影转过身。身后面是空的。

### 示例二：搬家之后

- 原句：搬家后他发现少了一样东西，那是很重要的回忆
- 改写：箱子堆满了客厅。厅里唯一没动的是那把椅子。椅子上还搭着她的围巾。围巾上的洗衣液味道已经散了。散了就再也闻不到了。到了现在他才发现，丢了的不是围巾。

### 示例三：深夜电话

- 原句：她半夜接到电话，知道出事了，整个人都慌了
- 改写：手机震了一下。下意识地翻身去看。看见来电显示的名字。名字亮了三秒就灭了。灭掉之后屏幕映出她自己的脸。脸上没有表情，只有泪痕。

## 关键提示

技巧藏起来，情绪露出来——咬尾是为了推情绪，不是为了炫技。
