---
name: 约喻
description: 让多个不同事物共享同一个喻体，以聚合之力揭示万物共通的本质
category: 修辞
---

## 核心原理

约喻让多个不同的本体共同归属于同一个喻体，不是"A像B"，而是"A、B、C统统都是D"。普通比喻是发散的，一个本体指向一个喻体，强调的是两者之间的独特相似。约喻恰恰相反，它是聚合的：将天南海北、看似无关的事物强行纳入同一个比喻框架，过滤掉枝节与差异，只留下最关键的共同本质。当读者发现这些迥异的事物竟然可以被同一个词统一时，会瞬间产生"原来如此"的认知冲击，这种发现式的共鸣远比被直接告知来得深刻。

## 使用方法

1. 从多个不同事物中抽象出一个共通特质，作为喻体
2. 将这些事物用同一个比喻框架串联起来，形成排布
3. 喻体本身要有足够的概括力和情感浓度，能承载多个本体的分量
4. 本体之间的差异越大，约喻的冲击力越强——越不相关的事物被统一，越有顿悟感

## 示例

### 示例一：写人生不同阶段的成长

- 原句：人在不同年纪有不同的追求，年轻时有壮志，中年有担当，老了就豁达了
- 改写：年少时的壮志、中年时的担当、暮年时的豁达，都是生命在不同海拔处的呼吸

### 示例二：写一种挥之不去的遗憾

- 原句：不管做什么事，他都会想起那次失误，喝茶会想起，走路会想起，睡觉前也会想起
- 改写：倒茶时想起，走路时想起，深夜闭眼前还是想起——遗憾是一杯永远续不满的空杯

### 示例三：写多种孤独的形态

- 原句：一个人吃饭是孤独的，一个人旅行也是孤独的，一个人住在大房子里更孤独
- 改写：独坐空桌的沉默、异乡街头的徘徊、满屋灯火里的无声，都是同一种孤独换上了不同的面具

## 关键提示

普通比喻是发散的——一个本体对一个喻体；约喻是聚合的——万物殊途同归于一喻，"原来如此"的共鸣瞬间爆发。
