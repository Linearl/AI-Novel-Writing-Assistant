## 核心原理

列锦将几个名词或名词短语直接摆在一起，不用动词、连词甚至形容词来连接。这种手法利用了人脑天然的"完形填空"本能——当一组意象被并列呈现而缺少逻辑连接时，读者的注意力会被强制激活，主动在空白处补全画面、情绪和因果关系。这种由读者自行"发现"的参与感远超被动接收，使文字以极简的形式承载极浓的情感密度。列锦越克制，留白越大，读者能注入的个人经验就越多，共鸣也就越深。

## 使用方法

1. 写情绪时不直接写出情绪词，只选取能承载情绪的具象物品
2. 将选好的名词去掉所有动词和连接词，逐个罗列
3. 每个意象之间用逗号或空格隔开，不做任何解释
4. 意象的排列顺序要有情绪递进的节奏，从表层到深层、从外围到核心

## 示例

### 示例一：写独居老人的下午

- 原句：老人一个人在家坐了一下午，椅子很旧，茶也凉了，显得很冷清
- 改写：一个下午，一把藤椅，一壶凉透的茶，一只停摆的挂钟

### 示例二：写分手后的空房间

- 原句：她搬走了，屋子里空荡荡的，只剩下一些她的痕迹
- 改写：衣柜空出一半，衣架还在，窗台的绿植枯了，枕头上还有一根长发

### 示例三：写战后废墟

- 原句：村庄被战火摧毁了，到处都是破碎的瓦片和倒塌的墙壁
- 改写：碎瓦，断梁，半面墙，墙上还贴着一张年画

## 关键提示

列锦适合放在段落结尾或情绪需要收住的时刻——前面的情绪被压缩进几个名词里，读者读到那里会自己拆开、自己释放。
