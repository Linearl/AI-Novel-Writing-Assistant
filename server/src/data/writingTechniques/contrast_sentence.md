## 核心原理

落差句的核心机制是"搭建期待然后亲手拆毁"。前半段用具体的细节、真实的付出过程铺陈得热气腾腾，读者会跟着你一起构建那个美好的期待，一起投入感情。然后结尾用最轻描淡写的一句话，不动声色地把全部温度抽走。这种巨大的心理落差比任何直白的悲伤描写都更扎心，因为读者不只是在看角色的痛苦，而是在亲身经历那种"从满怀希望到瞬间落空"的失重感。落差句的底层逻辑是：伤害的强度不取决于事件本身，而取决于期待与现实之间的距离。

## 使用方法

1. 铺垫要够"实"：用具体的细节和真实的付出过程来构建期待，让读者确信"这事能成"
2. 中间可以制造一些小胜利，让期待感层层加码
3. 转折要够"轻"：用最平淡、最日常的一句话收尾，语气不带任何戏剧性
4. 落点背后要藏一个真实的生活真相，让读者在平静中感受到钝痛

## 示例

### 示例一：三年异地恋

- 原句：我和他分手了，三年的感情就这么结束了，很伤心
- 改写：我们熬过了三年异地恋，一起规划过未来的小家，连阳台要种什么花都定好了。最后分手，只是因为他轻描淡写一句"我妈不同意，算了吧"。

### 示例二：职场晋升

- 原句：他努力了两年却没有升职，非常失望
- 改写：连续两年绩效第一，加班到凌晨的夜晚他自己也数不清了。新来的主管入职第一天，在群里说"以后大家有什么问题直接找我"，@的是另一个人的工号。

### 示例三：久别重逢

- 原句：多年后重逢，她发现他已经变了，感到很失落
- 改写：她攒了两年的勇气才发出那条消息，他秒回了一个"在"。她打了三段话删掉，最后只发了一句"你最近好吗"。他隔了十分钟回：挺好的，你呢。三个字后面没有标点。

## 关键提示

转折越轻描淡写，与厚重铺垫的对比越扎心。
