category: 叙事

## 核心原理

一个意象第一次出现时美好、温暖、充满希望。当它再次出现时，语境已天翻地覆——同样的画面变成了一把刀。这种反杀不是作者硬塞的，是读者自己拿着前面的"蜜糖"亲手经历了它变成"砒霜"的过程，是从心里长出来的痛。意象反杀之所以格外刺痛，是因为读者投入了真实的情感去喜爱那个意象，当它的含义反转时，读者要亲手撕碎自己的那份喜爱——这种痛来自内部，不是外部强加的。

## 使用方法

1. 给意象"埋种子"：在前面种下一个不起眼的日常细节，等悲剧发生后让它破土而出
2. 让意象"变脸"：同一个物品前半段承载温暖情感，后半段语境突变自动翻转成相反含义
3. 用意象承载"真相"：选一个意象作为真相容器，让前面看似赞美的话在真相揭晓时变成刀子
4. 让意象在关键时刻"缺席"：不写出反转，而是让读者发现那个意象不再出现了

## 示例

### 示例一：小手

- 原句："有庆死后，他非常想念儿子。"
- 改写："我背着他走在路上，有庆的小手一甩一甩的，像是在指挥我。"（初读温暖，得知有庆死后回看——那个指挥再也没有方向了。读者自己给记忆补上了那一刀。）

### 示例二：糖炒栗子

- 原句："妻子去世后，他路过卖栗子的摊位，想起了从前。"
- 改写："以前每次路过这个摊位，她都要买一袋。"他站在摊前，老板问"还是老样子？两袋？"他张了张嘴。"一袋。"老板没多问，多装了一些。（"一袋"两个字，比任何悼亡都重。"多装了一些"是老板没说出口的安慰。意象没有变，变的是旁边少了一个人。）

### 示例三：栀子花

- 原句："她离开后，他看到她种的栀子花，十分伤感。"
- 改写："阳台上那盆栀子花又开了。她走之前说过：'这花好养，浇浇水就行。'他浇了一年的水。花很好，开了三轮。他忽然意识到，她说的"好养"，是说给他听的——她知道自己不会在了。（栀子花从"好养"变成了"她提前安排好的告别"，同一个意象在两种理解下，一个是日常，一个是心碎。）"

## 关键提示

意象反杀的关键在于前后两次出现的意象完全相同，只改变语境。前面笑得越开心，后面哭得越惨烈。
