## 核心原理

白描就是用最朴素、最精炼的文字直接勾勒事物形态，不加任何主观形容词、比喻或情感渲染。它像一幅素描——只用名词和动词，不写"她很美"，只写"她穿着褪色的蓝布衣，安静地坐在门槛上"；不写"他很孤独"，只写"屋里就他一个人，桌上摆着一副碗筷"。让事物自己"说话"，读者自己会感受到。

## 使用方法

1. 用名词和动词，少用形容词和副词：如"枯藤老树昏鸦"三个名词就是一个画面
2. 不加主观评价：不告诉读者该怎样感受，只呈现他看到了什么、听到了什么
3. 情感全在细节里：把情绪翻译成可观察的动作和物件，让读者自己读出情绪
4. **白描+留白双剑合璧**：白描负责"锚定情绪"——用一个具体尖锐的细节把读者拉进场景；留白负责"放大情绪"——不说破结局不定义感受，让读者用自己的人生经验去补全故事。三步实操：选一个"带刺的细节"→做一次"减法表达"（删掉所有形容词和情绪词，只留名词和动词）→留一个"开放式结尾"（不要给结论不要给答案，让句子停在"未完待续"的地方）

## 示例

### 示例一：写伤心

- 渲染写法：她很伤心，眼泪止不住地流下来，窗外的雨声更显得凄凉
- 白描：她转过身去，肩膀轻轻抖着。雨打在玻璃上，模糊了外面的街灯

### 示例二：写穷人家凑钱（鲁迅《药》）

- 改写：华大妈在枕头底下掏了半天，掏出一包洋钱，交给老栓，老栓接了，抖抖地装入衣袋，又在外面按了两下

### 示例三：写冬夜等车

- 改写：他看了看表，把拉链拉到最顶端，两只手缩进口袋，身体微微缩着

### 示例四：白描+留白写爱而不得

- 白描细节："他发的最后一条朋友圈，我点了赞，又取消了。"
- 留白空间：没有写"我还爱他"，但"点赞又取消"的犹豫藏着没说出口的拉扯和不甘

### 示例五：白描+留白写父母老去

- 白描细节："我妈把我买的进口钙片，和她的降压药放在了同一个药盒里。"
- 留白空间：没有写"妈妈老了"，但"钙片和降压药放在一起"的画面藏着她悄悄接受身体变化的温柔与心酸

### 示例六：白描+留白写深夜孤独

- 白描细节："煮了一碗面，吃了三口，剩下的汤，我倒在了花盆里。"
- 留白空间：没有写"我好孤单"，但"吃不完的面"和"浇花的汤"藏着一个人吃饭时的漫不经心和无处安放的情绪

## 关键提示

白描不是写得少，而是每个字都精准。只用"谁＋做了什么＋什么样（不写情绪词）"来描述。不写"冷"，但读者能感到冷；不写"焦急"，但读者知道他在赶时间。克制的高级感——不煽情反而更动人。
