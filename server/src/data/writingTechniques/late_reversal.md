category: 叙事

## 核心原理

前半句故意写得很俗、很笨、很让人失望，读者心里已经在嘀咕"这什么水平"。后半句突然拔地而起，把前面所有的平庸瞬间翻盘。这种反转不是靠华丽词藻硬撑的，而是靠"落差"自己长出来的力量——前面压得越低，后面弹得越高。逆挽的核心机制是管理读者预期：先让读者以为故事在低处，再把他们一把抛向高空，那股失重感就是文字的冲击力。

## 使用方法

1. 先俗后雅：前半句用最土最直白的话起笔，后半句用雅致辽阔的意象收束
2. 先小后大：从极小极无聊的细节入手，后半句突然把格局拉到人生、命运的高度
3. 先问后答：前半句抛一个幼稚或莫名其妙的问题，后半句给出让人沉默的答案
4. 先泄后聚：前半段用散乱、琐碎、毫无章法的笔触，后半段用一个画面把所有碎片收拢成一条线

## 示例

### 示例一：鸡鸣

- 原句："一只公鸡在叫，声音很大。"
- 改写："鸡叫一声撅一撅，鸡叫两声撅两撅。三声唤出扶桑日，扫退残星与晓月。"（前两句粗俗得让群臣面面相觑，后两句瞬间把全诗拉到天地苍穹的高度。前面的"撅一撅"不再是笑话，而是蓄势。）

### 示例二：菜市场

- 原句："她每天去菜市场买菜，觉得很无聊。"
- 改写："白菜八毛，豆腐一块二，鸡蛋论斤不论个。她提着菜往回走，路过一棵玉兰树。花开了。她站了一会儿。丈夫去世后的第一个春天，她第一次注意到花开了。（菜市场的琐碎铺垫不是废话，是她生活的全部底色。当"花开了"出现时，前面所有的庸常都变成了反衬——活着这件事，本身就是从八毛钱的白菜里长出来的。）"

### 示例三：童年趣事

- 原句："小时候他和朋友们在河边玩水，很开心。"
- 改写："那年夏天我们四个人在河里摸了一下午的鱼，一条没摸着，裤子湿了三条。后来我们坐在河边把脚晒干。现在他们三个都在不同的城市，我坐在这条干了的河边，脚还是湿的。（前面是童年趣事的俗套开场，最后那句"脚还是湿的"把时空折叠——干了的河与湿着的脚，过去和现在砸在一起，笑意还没收完就变成了泪意。）"

## 关键提示

逆挽的本质是"先把你骗进门，再把你轰上天"。精彩的表达是等来的，逆挽是抢来的——先踩你一脚，再扶你起来。
