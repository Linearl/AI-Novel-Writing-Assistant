# REQ-2005 步骤6新建章节删除与误操作恢复

> 目标版本：v0.1
> 状态：⏳ 进行中

---

## 1. 任务概述

### 1.1 需求来源

用户反馈：步骤6"新建章节"按钮创建空壳章节后，无法删除，误操作后无法恢复。

### 1.2 核心内容

1. Chapter 表新增 `deletedAt` 软删除字段
2. 后端提供删除/恢复 API
3. 前端在选中章节后显示删除按钮 + 确认对话框
4. 删除后提供 10 秒 undo toast

### 1.3 前置条件

- Prisma migration 支持
- 现有 toast 组件支持自定义操作
