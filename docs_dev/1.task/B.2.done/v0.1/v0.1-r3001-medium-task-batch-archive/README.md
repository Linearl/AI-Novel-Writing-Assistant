# REQ-3001 任务中心多选与批量归档

> 创建日期：2026-06-26
> 目标版本：v0.1
> 状态：⏳ 进行中

---

## 1. 任务概述

### 1.1 需求来源

用户反馈：任务中心的任务列表目前只支持逐个选中和逐个归档，当积累了大量已完成或失败的历史任务时，需要反复"点击任务 → 查看详情 → 点击归档"才能清理。缺少多选和批量操作能力，导致任务清理效率低。

### 1.2 核心内容

1. 任务列表每一项增加 Checkbox 多选控件，支持独立勾选/取消
2. 列表顶部增加工具栏，提供"全选/取消全选"快捷操作和已选计数
3. 工具栏增加"批量归档"按钮，一键归档所有已选任务
4. 后端新增 `POST /api/tasks/batch-archive` 批量归档端点，支持单次请求归档多个任务

### 1.3 前置条件

- 任务中心页面（`TaskCenterPage`）已稳定
- 单任务归档 API `POST /api/tasks/:kind/:id/archive` 已存在
- `ARCHIVABLE_STATUSES` 已定义可归档状态集合（succeeded / failed / cancelled）
