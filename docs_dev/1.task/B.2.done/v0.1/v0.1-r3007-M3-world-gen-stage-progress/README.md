# REQ-3007 世界生成阶段进度显示

> 创建日期：2026-06-29

> 状态：📋 待办

## 1. 任务概述

### 1.1 需求来源

用户使用体验反馈：世界生成向导步骤 2（骨架生成）按钮在生成过程中仅显示"生成中"等笼统文案，用户无法感知当前进展，体验焦虑感强。

### 1.2 核心内容

1. 后端将骨架生成从单次 LLM 调用拆分为多阶段流水线，每阶段发送 SSE 进度事件
2. 前端按钮动态显示当前阶段名称和进度（如"正在生成核心规则... 2/7"）
3. 生成过程中提供有意义的阶段标签，而非通用的"生成中"

### 1.3 前置条件

- 无
