## 1. 背景与问题

自动导演任务在创建时设定的多项高级参数（是否自动审校、是否自动修复、审批点配置、artifactSyncMode）一旦确认即锁定于 `seedPayload` 中。任务暂停后，用户只能调整运行时策略参数，无法修正创建时设置不当的高级参数。

用户痛点：创建时对参数影响认知不足，设错后无法纠正；task 中心缺少编辑入口。

## 2. 目标

1. 暂停后，任务中心提供"编辑高级设置"面板，复用创建时的高级设置组件
2. 后端支持暂停态覆写 autoExecutionPlan 部分字段和审批点配置
3. 前端暴露 modelTier 和 artifactSyncMode

## 3. 变更范围

- 后端：新增/扩展 API 支持暂停后修改
- 前端：任务中心增加编辑按钮 + 模态面板；创建时增加 artifactSyncMode；运行时策略卡增加 modelTier
