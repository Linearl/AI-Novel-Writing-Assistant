## 1. 任务概述

### 1.1 需求来源

自动导演全自动模式能力诊断报告（`docs_dev/3.analysis/diagnosis/01-active/2026-06-26-自动导演全自动模式能力诊断.md`）。

### 1.2 核心内容

1. 暂停后允许调整是否自动审校（autoReview）和是否自动修复（autoRepair）
2. 暂停后允许调整审批点配置（复用创建时高级设置面板）
3. 前端暴露模型档次（modelTier）选择器
4. 前端暴露 artifactSyncMode 选择器（创建时 + 暂停后均可配）
5. 任务中心详情页增加"编辑高级设置"按钮作为调整入口

### 1.3 前置条件

- 无（纯增量功能，不依赖其他 REQ）
