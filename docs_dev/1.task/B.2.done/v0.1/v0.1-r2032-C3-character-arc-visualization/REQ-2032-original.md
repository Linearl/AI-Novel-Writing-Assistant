## 1. 背景与问题

REQ-2031 提供了后端查询工具，但无前端可视化。用户需要直观看到角色弧光时间线和关系网络。

## 2. 目标与范围

- 角色弧光时间线图（事件节点 + 情绪曲线）
- 角色关系网络图（节点=角色，边=四维关系分数）
- 角色选择器，集成到角色管理页面
