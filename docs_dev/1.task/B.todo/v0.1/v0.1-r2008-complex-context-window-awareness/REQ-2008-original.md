## 1. 背景

系统完全不感知模型上下文窗口大小。`ModelRouteConfig` 只有 `maxTokens`（输出限制），无输入上下文窗口配置。超限时原始 API 错误直接传播。

## 2. 目标

1. 每个模型路由可配置上下文窗口（默认 1M，可取消勾选降级到 256K）
2. 使用率超过 85% 时自动压缩至 40% 以下
3. 覆盖 Prompt Registry、Chat、CreativeHub 三条路径

## 3. 验收标准

- [ ] 模型路由设置页显示 1M 上下文 Checkbox（默认勾选）
- [ ] 超 85% 自动压缩至 40%
- [ ] 三条路径均集成
