## 核心目标

1. 评估将 server 迁移到 ESNext + tsx 运行时的可行性（内存开销、启动时间、CJS 互操作）
2. 探索 TypeScript 5.9 Project References 替代路径别名（让 tsc 自动 build 依赖）
3. 将 prisma:generate 加入 server build 的 prebuild hook
