## 核心目标

1. 清点所有 `routes/` 下文件和 `modules/` 现有覆盖，产出迁移清单
2. 将仍活跃的 `routes/` 文件逐模块迁移到 `modules/` 对应子目录
3. 统一 `app.ts` 的路由导入规范（仅从 `modules/` 导入路由注册函数）
4. 制定新路由落位规则文档

## 受影响文件

- `app.ts` — 路由注册入口
- `routes/*` — 约 30 个路由文件
- `modules/*` — 各模块 http 入口

## 验收标准

- `app.ts` 只剩 `modules/` 导入
- typecheck 通过
- 路由测试全绿
