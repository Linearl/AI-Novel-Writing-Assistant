# REQ-2046 写法模板导出/导入

## 背景

写法模板市场是规模效应的关键路径。用户 A 用拆书功能提炼了一套写法，用户 B 想写同类型小说时可以直接引用。当前 StyleProfile 数据存储在数据库中，没有导出/导入机制。

## 需求描述

### 导出
- 风格管理页面每个 StyleProfile 旁新增"导出"按钮
- 导出为 JSON 文件，包含完整的 StyleProfile 数据
- 文件名格式：`{profile-name}-{date}.json`

### 导入
- 风格管理页面新增"导入"按钮
- 选择 JSON 文件后解析并预览
- 导入冲突处理：同名 → 覆盖/新建/跳过（用户选择）

## 验收条件

1. 导出的 JSON 可被导入还原
2. 导入的 StyleProfile 功能完整（含 anti-AI 规则）
3. 冲突处理不丢失现有数据
