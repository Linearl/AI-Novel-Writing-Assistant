## 需求背景

小说预览页面存在三个用户体验痛点：

1. **字号固定**：长时间阅读眼睛容易疲劳
2. **页宽不可调**：宽屏显示器上空间浪费
3. **字体/背景不可定制**：无法适应不同用户偏好

本任务通过在预览页面的标题和"复制正文"按钮之间添加控制面板，允许用户实时调节以下显示设置：

- 字号（12-36px）
- 字体（6种系统字体）
- 加粗（开关）
- 页宽（500-1200px）
- 背景色（4种预设主题）

## 核心功能

### 控制面板组成

```
┌──────────────────────────────────────────────────────┐
│  小说标题                                            │
│  ─────────────────────────────────────────────────── │
│  [控制面板]                                          │
│  ├── 字号：[12]----●----[36] px                      │
│  ├── 字体：[宋体 ▼]                                  │
│  ├── 加粗：[OFF]                                     │
│  ├── 页宽：[500]----●----[1200] px                   │
│  └── 背景：[白] [米黄] [绿] [黑]                     │
│  ─────────────────────────────────────────────────── │
│  [复制正文] [其他操作]                                │
│                                                      │
│  小说内容预览区域...                                  │
└──────────────────────────────────────────────────────┘
```

### 交互流程

1. 用户调整任意控制项
2. 立即反映到预览区域
3. 无需点击确认按钮
