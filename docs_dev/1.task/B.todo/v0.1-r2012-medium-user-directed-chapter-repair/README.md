# REQ-2012 用户指导式章节修复

> 创建日期：2026-06-26

> 状态：📋 待办

---

## 1. 任务概述

### 1.1 需求来源

用户在使用 auto-director 审计章节后发现，当前提供三种修复方案（一键补丁、重写整章、手动编辑器）不足以满足需求。用户希望能够**提出自己的修改方向**，让 AI 按照用户指示执行定向修改，而不是只能选择预设的修复策略或完全手动编辑。

### 1.2 核心内容

1. 在章节审计结果面板新增"按指导修复"入口，用户可输入自由文本修改方向
2. 后端修复接口扩展 `userInstruction` 字段，将用户指导注入修复流程
3. 修复执行时强制使用 `heavy_repair` 模式，确保 AI 以用户方向为最高优先级

### 1.3 前置条件

- 无外部依赖。基于现有 `POST /:id/chapters/:chapterId/repair` SSE 管线扩展
