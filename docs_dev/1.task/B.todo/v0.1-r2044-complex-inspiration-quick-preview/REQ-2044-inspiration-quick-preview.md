# REQ-2044 灵感页快速预览

## 背景

当前从灵感到看到第一段文字需要 10+ 分钟（多步配置 → AI 规划 → 拆章 → 细化 → 正文）。小白用户流失率极高。

## 需求描述

在创建页（NovelCreate）内部新增快速预览模式：
1. 用户输入灵感（一句话或一段描述）
2. 点击"快速预览"
3. AI 生成 3 个方向候选：标题 + 梗概 + 前 500 字预览
4. 用户选定一个方向
5. 可选：继续生成前 3 章
6. 满意后一键转入正式自动导演流程

## 验收条件

1. 从输入灵感到看到预览 < 30 秒
2. 3 个方向候选质量可读（不是乱码/重复）
3. 转入正式流程后，预览内容不丢失
4. 不影响现有的正式开书流程
