## AI 接管时的素材处理流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    AI 接管模式 - 素材处理流程                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  用户点击"AI 接管"按钮                                                    │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  NovelAutoDirectorDialog                                         │      │
│  │  ├─ 选择运行模式（单步/全书自动）                                │      │
│  │  ├─ 选择起始步骤                                                 │      │
│  │  └─ 【新增】导入素材按钮                                         │      │
│  │      └─ 打开 MaterialParseDialog                                 │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  启动 auto_director 任务                                         │      │
│  │                                                                   │      │
│  │  传递参数：                                                       │      │
│  │  {                                                                │      │
│  │    novelId,                                                       │      │
│  │    lane: "auto_director",                                         │      │
│  │    seedPayload: {                                                 │      │
│  │      basicForm: {                                                 │      │
│  │        worldSetting: "...",   // 从导入的素材中提取              │      │
│  │        characters: "...",     // 从导入的素材中提取              │      │
│  │        outline: "...",        // 从导入的素材中提取              │      │
│  │      }                                                            │      │
│  │    }                                                              │      │
│  │  }                                                                │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  Auto Director 后端处理                                          │      │
│  │                                                                   │      │
│  │  检查 seedPayload 中是否有素材：                                 │      │
│  │  if (basicForm.worldSetting ||                                    │      │
│  │      basicForm.characters ||                                      │      │
│  │      basicForm.outline) {                                         │      │
│  │                                                                   │      │
│  │    // 情况1：有素材                                               │      │
│  │    // ├─ 保存到 Novel 表                                         │      │
│  │    // ├─ 同步到 StoryMacroPlan.storyInput                        │      │
│  │    // └─ 使用素材作为上下文注入到各步骤                          │      │
│  │                                                                   │      │
│  │  } else {                                                         │      │
│  │                                                                   │      │
│  │    // 情况2：无素材                                               │      │
│  │    // ├─ AI 生成初始规划（从零开始）                             │      │
│  │    // ├─ 生成世界观、角色、大纲                                  │      │
│  │    // └─ 用户确认后再继续                                        │      │
│  │                                                                   │      │
│  │  }                                                                │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  执行后续步骤（步骤1-7）                                         │      │
│  │                                                                   │      │
│  │  所有步骤都能读取到：                                             │      │
│  │  ├─ Novel.worldSetting                                           │      │
│  │  ├─ Novel.characters                                             │      │
│  │  ├─ Novel.outline                                                │      │
│  │  └─ StoryMacroPlan.storyInput                                    │      │
│  │                                                                   │      │
│  │  因此：                                                           │      │
│  │  ├─ 步骤2（故事宏观规划）会自动使用 storyInput                   │      │
│  │  ├─ 步骤4-5（卷战略）会读取 outline（我们之前修改的）            │      │
│  │  └─ 所有步骤都能访问完整素材                                     │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 素材同步机制

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         素材同步机制详解                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  导入/修改素材时（任何入口）                                     │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  API: POST /novels/:id/material/sync                             │      │
│  │                                                                   │      │
│  │  输入：                                                           │      │
│  │  {                                                                │      │
│  │    worldSetting: "修改后的世界观...",                             │      │
│  │    characters: "修改后的角色...",                                 │      │
│  │    outline: "修改后的大纲...",                                    │      │
│  │    options: {                                                     │      │
│  │      syncToStoryInput: true,    // 是否同步到 storyInput         │      │
│  │      regenerateStoryEngine: false // 是否重新生成故事引擎        │      │
│  │    }                                                              │      │
│  │  }                                                                │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  同步逻辑                                                        │      │
│  │                                                                   │      │
│  │  1. 更新 Novel 表                                                │      │
│  │     UPDATE novel SET                                             │      │
│  │       worldSetting = :worldSetting,                              │      │
│  │       characters = :characters,                                  │      │
│  │       outline = :outline                                         │      │
│  │     WHERE id = :novelId                                          │      │
│  │                                                                   │      │
│  │  2. 重新构建 storyInput                                          │      │
│  │     storyInput = worldSetting + "\n\n"                           │      │
│  │                 + characters + "\n\n"                             │      │
│  │                 + outline                                         │      │
│  │                                                                   │      │
│  │  3. 更新 StoryMacroPlan                                          │      │
│  │     UPSERT INTO story_macro_plan                                 │      │
│  │       (novelId, storyInput)                                      │      │
│  │     VALUES (:novelId, :storyInput)                               │      │
│  │     ON CONFLICT (novelId) DO UPDATE                              │      │
│  │       SET storyInput = :storyInput                               │      │
│  │                                                                   │      │
│  │  4. 可选：重新生成故事引擎                                       │      │
│  │     if (regenerateStoryEngine) {                                 │      │
│  │       await regenerateStoryMacroPlan(novelId, storyInput);       │      │
│  │     }                                                            │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│      │                                                                      │
│      ↓                                                                      │
│  ┌──────────────────────────────────────────────────────────────────┐      │
│  │  后续步骤自动获得更新后的素材                                     │      │
│  │                                                                   │      │
│  │  步骤2（故事宏观规划）：                                         │      │
│  │  ├─ 从数据库读取 StoryMacroPlan.storyInput                      │      │
│  │  └─ 使用更新后的 storyInput 生成故事引擎                         │      │
│  │                                                                   │      │
│  │  步骤4-5（卷战略）：                                             │      │
│  │  ├─ 从数据库读取 Novel.outline                                  │      │
│  │  └─ 注入到 prompt 中（我们之前修改的代码）                      │      │
│  │                                                                   │      │
│  │  其他步骤：                                                       │      │
│  │  └─ 读取 Novel 表的 worldSetting, characters, outline            │      │
│  └──────────────────────────────────────────────────────────────────┘      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│  │  │   └─ 进入步骤0 → 步骤1-7            │                                │
│  │  │                                      │                                │
│  │  └─ AI 接管                             │                                │
│  │      └─ 启动 auto_director             │                                │
│  └────────────────────────────────────────┘                                │
│                    ↓                                                        │
│  决策点3：AI 接管时是否有素材？                                            │

│  │  ├─ worldSetting/characters/outline    │                                │
│  │  │                                     │                                │
│  │  ├─ 有素材                             │                                │
│  │  │   ├─ 保存到 Novel 表               │                                │
│  │  │   ├─ 同步到 StoryMacroPlan         │                                │
│  │  │   └─ 使用素材执行步骤1-7          │                                │
│  │  │                                     │                                │
│  │  └─ 无素材                             │                                │
│  │      ├─ AI 生成初始规划               │                                │
│  │      ├─ 生成世界观、角色、大纲       │                                │
│  │      └─ 用户确认后继续               │                                │
│  └────────────────────────────────────────┘                                │
│                    ↓                                                        │
│  决策点4：是否需要重新生成故事引擎？                                       │
│  ┌────────────────────────────────────────┐                                │
│  │ 用户修改素材后                         │                                │
│  │  ├─ 是 → 调用 POST /novels/:id/       │                                │
│  │  │         story-macro/constraint/build│                                │
│  │  │         重新生成故事引擎            │                                │
│  │  │                                     │                                │
│  │  └─ 否 → 只同步到 storyInput          │                                │
│  │           不重新生成故事引擎           │                                │
│  └────────────────────────────────────────┘                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│  └─ 数据库                                                                  │
│      └─ 确保 StoryMacroPlan.storyInput 字段存在且可用                      │
│                                                                             │
│  优先级2（重要）- 3天                                                       │
│  ├─ 步骤0 UI                                                               │
│  │   ├─ 在 NovelCreate 中添加"导入素材"按钮                               │
│  │   ├─ 在 NovelEdit 步骤0 中添加素材管理界面                             │
│  │   └─ 素材预览和编辑功能                                                 │
│  │                                                                          │
│  ├─ AI 接管对话框集成                                                      │
```
