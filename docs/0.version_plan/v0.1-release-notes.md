## 1. 版本概述

v0.1.00 是本 fork 相对上游 v0.32.0 的首个正式定版。上游已提供完整的 AI 小说创作工作台基础架构，本版本在此基础上完成 **110 个任务包**，覆盖创作辅助、自动导演增强、章节管理、UI 交互、运维支持、架构重构、安全加固、测试工程和代码清理。

### 与上游的关系

- 上游 v0.32.0 提供了完整的单书工作流（从开书到章节执行）
- 本 fork 专注于**审校质量提升**、**角色系统增强**、**运维可观测性**和**工程化治理**
- 漫画/短剧模块已在本 fork 中移除（与小说创作方向无关，r7013）
