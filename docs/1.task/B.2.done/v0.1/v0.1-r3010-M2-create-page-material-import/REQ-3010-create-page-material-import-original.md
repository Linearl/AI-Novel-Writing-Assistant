# REQ-3010 创建页素材导入

## 背景

当前创建页是固定表单，用户需要逐项填写。已有世界观、角色、大纲素材的用户无法快速导入，只能手动复制粘贴到每个字段。

## 需求描述

在创建页新增"粘贴素材"功能：
1. 用户点击"粘贴素材"按钮
2. 弹出大文本输入框
3. 用户粘贴已有素材（世界观、角色、大纲、灵感等任意格式）
4. AI 自动识别内容类型并拆分到对应字段
5. 拆分结果可预览和手动调整
6. 确认后自动填入创建表单

## 验收条件

1. AI 拆分准确率 > 80%（正确识别标题、简介、世界观、角色等）
2. 拆分结果可预览
3. 用户可手动调整拆分结果
4. 确认后正确填入对应表单字段
