## 1. 任务概述

### 1.1 需求来源

用户需求讨论：当前 AI 润色增强完全不接入写法引擎（StyleEngine），且缺少正向的"文笔技法"指导机制。用户希望新增一个"文笔资料库"，以可开关的写作技法集合形式，配合 AI 自动筛选，在润色增强时注入合适的技法指导。

### 1.2 核心内容

1. **文笔资料库数据层**：MD 文件存储技法（frontmatter 两层加载），DB 自动同步开关状态，三级池子（全局 ∪ 画像 ∪ 小说）
2. **文笔资料库管理页面**：新增 `/writing-techniques` 路由页面，管理全局开关、浏览技法内容
3. **画像级绑定**：在写法引擎画像编辑器中新增"文笔技法"面板
4. **润色增强接入 StyleContract**：修复现有 styleSummary gap，将完整的 StyleContract（画像规则 + 反 AI 规则）注入章节编辑器改写 prompt
5. **AI 技法筛选 + 注入**：润色时组装候选池 → AI 两阶段筛选（先看概述选技法，再加载全文注入改写）
6. **批量导入**：从 `temp/写法学习` 批量提取 40+ 篇素材，AI 精炼为结构化技法 MD 文件

### 1.3 前置条件

- 无硬性前置依赖。StyleEngine、AntiAiRule、章节编辑器均已就绪。
