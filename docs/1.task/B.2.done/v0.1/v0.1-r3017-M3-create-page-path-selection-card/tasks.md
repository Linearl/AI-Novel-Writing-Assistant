### 阶段一：路径选择卡片UI（1天）

#### T1.1 设计路径选择卡片组件

**任务**：创建 PathSelectionCard 组件

**工作内容**：
- 创建 `client/src/pages/novels/components/PathSelectionCard.tsx`
- 定义组件 props 接口（selectedPath、onSelectPath 等）
- 实现三个路径选项的 UI 布局
- 添加选中状态的视觉反馈（边框高亮）
- 添加响应式样式

**DoD**：
- [ ] 组件文件创建完成
- [ ] Props 接口定义清晰
- [ ] 三个路径选项 UI 显示正常
- [ ] 选中状态有视觉反馈

**测试**：
- [ ] 单元测试：路径点击回调正确触发
- [ ] 单元测试：选中状态样式正确应用

- 添加 selectedPath 状态管理

**DoD**：

**测试**：
- [ ] 集成测试：路径选择后动态内容正确切换
