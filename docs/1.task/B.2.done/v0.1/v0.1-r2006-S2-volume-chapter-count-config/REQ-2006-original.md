# REQ-2006 步骤4每卷章节数手动调整

> 状态：⏳ 进行中（完成后改为 ✅ 完成）

## 需求元信息

| 字段 | 内容 |
| ---- | ---- |
| 需求编号 | REQ-2006 |
| 优先级 | P1 |
| 来源 | 用户需求 — 当前每卷章节数由系统自动分配，用户无法手动调整 |
| 分类 | 2xxx 核心功能开发 |
| 复杂度 | simple |

---

## 1. 背景

当前每卷章节数由 `allocateChapterBudgets()` 自动分配（均匀或按比例加权），用户无法手动调整。

---

## 2. 目标

用户在步骤4卷卡片中可手动设置每卷目标章节数，手动值优先于自动分配。

---

## 3. 验收标准

- [ ] 卷卡片中显示章节数输入框
- [ ] 手动值优先于自动分配
- [ ] 清除后恢复自动分配
- [ ] 节奏板生成使用手动值
