# REQ-2006 决策日志

## D1: 字段为 optional nullable 而非默认值

| 字段 | 内容 |
| ---- | ---- |
| 日期 | 2026-06-27 |
| 决策人 | AI |
| 决策 | `targetChapterCount` 为 `number | null | undefined`，null/undefined 表示使用自动分配 |
| 原因 | 区分"用户明确设为自动"和"用户设了具体值"两种状态。避免用 0 或 -1 等魔法值表示"自动"。 |
| 影响 | 前端输入框需要处理空值情况 |

## D2: 不自动重算其他卷

| 字段 | 内容 |
| ---- | ---- |
| 日期 | 2026-06-27 |
| 决策人 | AI |
| 决策 | 用户调整一卷的章节数后，不自动调整其他卷 |
| 原因 | 用户手动调整通常有明确意图，自动重算可能破坏其规划。总预算超出时仅提示。 |
| 影响 | 手动值总和可能与总预算不一致 |
